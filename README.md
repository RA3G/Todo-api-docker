Todo API – Dockerized & Deployed on AWS

A containerized Todo REST API built with Node.js, MongoDB, and Docker.
Deployed on AWS EC2 with Docker Compose.

 Tech Stack
•	Node.js
•	Express
•	MongoDB
•	Docker
•	Docker Compose
•	AWS EC2 (Cloud Deployment)

  RUN LOCALLY ON DOCKER 
git clone https://github.com/RA3G/Todo-api-docker.git


cd Todo-api-docker

RUN CONTAINERS 
docker-compose up --build

Application deployed on AWS EC2 instance using Docker Compose.
